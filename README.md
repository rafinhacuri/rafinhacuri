## 👋 About Me

I'm **Rafael Curi**, a **Software Engineer, DevOps Engineer, and Full Stack Developer** focused on building secure, scalable, and high-performance systems.

Currently working at the **Brazilian Center for Physics Research (CBPF)**, where I develop modern web applications, APIs, internal platforms, and infrastructure solutions.

I work across the full software lifecycle — from frontend interfaces and backend services to production deployments, automation, DNS infrastructure, authentication, and server administration.

---

## 🚀 What I Do

- ⚡ Build modern applications with **Nuxt, Vue, TypeScript, and Go**
- 🏗️ Design backend services using **MongoDB, PostgreSQL, Redis, and REST APIs**
- 🔐 Work with authentication, authorization, **LDAP, SSO, OAuth, and RBAC**
- 🌐 Manage DNS infrastructure using **PowerDNS, DNSSEC, and Nginx**
- 🐳 Deploy and maintain services with **Docker, Linux, CI/CD, and automation**
- 📈 Improve performance, reliability, observability, and developer experience
- 🎓 Computer Science student passionate about software architecture and infrastructure

---

## 🎯 Current Focus

- 🚀 Building **SanchezDNS**, a modern DNS management platform
- 🛡️ Developing secure systems with strong access control and infrastructure automation
- ⚙️ Improving deployment workflows, monitoring, and service reliability
- 📚 Learning more about distributed systems, networking, cybersecurity, and backend scalability
